ajouterLivraison

(
  • idIntersection
  • idLivraison
  • idClient
)

Defined in controleur\controleur.js:341

Affiche l'écran de chargement puis appelle le service /controleur/ajouter-livraison qui permet d'ajouter une livraison.
Si l'appel au service réussit, càd que l'ajout s'est bien déroulé, la méthode Controleur/_ajouterLivraisonOk:method est appelée ( _ajouterLivraisonErr dans le cas contraire).

Parameters:

  • idIntersection Object

    ID de l'intersection à laquelle on veut livrer.

  • idLivraison Object

    ID de la livraison devant précéder celle que l'on crée.

  • idClient Object

    ID du client à livrer.
